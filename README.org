* About

Poor efficiency calculator demo, using *regexp* and *cmd/yacc*, use *regexp* to simulate *lex*.

The goal of this repository is *making syntax simple*, without considering efficiency.
* Example
#+BEGIN_SRC sh
  > 3*2.2**10
  7967.976837
  > 2^(8+1)
  512.000000
  > ExIt
  Quit now ~
#+END_SRC
